# 📌 OCaml шпаргалка операторов

## 🔢 Арифметика

```ocaml
+   сложение
-   вычитание
*   умножение
/   целочисленное деление
mod остаток от деления
```

Примеры:

```ocaml
3 + 2 = 5
5 - 1 = 4
4 * 2 = 8
7 / 2 = 3
7 mod 2 = 1
```

---

## ⚠️ Сравнение

```ocaml
=    равно
<>   НЕ равно
<    меньше
>    больше
<=   меньше или равно
>=   больше или равно
```

Примеры:

```ocaml
3 = 3      true
3 <> 4     true
5 < 2      false
```

---

## 🔤 Логика

```ocaml
&&   И (and)
||   ИЛИ (or)
not  НЕ
```

Примеры:

```ocaml
true && false   = false
true || false   = true
not true        = false
```

---

## 📦 Работа со строками

```ocaml
String.length s        длина строки
String.get s i         символ по индексу
```

Пример:

```ocaml
String.get "abc" 1   = 'b'
```

---

## 🔁 Базовые идеи (очень важно)

```ocaml
i + 1   → следующий индекс
i - 1   → предыдущий
```

---

## 📌 Функции

```ocaml
let f x = ...     обычная функция
fun x -> ...      анонимная функция (lambda)
```

Пример:

```ocaml
fun c -> c >= '0' && c <= '9'
```
